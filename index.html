CSS Outline Offset

This property adds space between an outline and the edge or border of an element.

Outline differs from borders! Unlike borders, the outline is drawn outside the element's border and may overlap other content. Outline is not affected by the element's dimension since it;s not a part of an element.

The following examples use the outline offset property to add space between the border and the outline

CSS 2D Transforms

        CSS transforms allow you to move, rotate, scale, and skew elements.
        Follwing are the 2d tranfromation methods used with CSS transform property:
        (a) translate()
        (b) rotate()
        (c) scaleX()
        (d) scaleY()
        (e) scale()
        (d) skewX()
        (e) skewY()
        (d) skew()
        (f) matrix()
